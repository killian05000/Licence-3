###############COMPILATEUR L###############
Pour utiliser le compilateur,dans un terminal
situé dans le dossier courant utilisez :
$ make
$ ./compilo X Y

X étant un paramètre il peut prendre plusieurs valeurs
et lancer différentes opérations :
-l : affichage analyse lexcial
-s : analyse syntaxique (aucune sortie si OK, sinon erreur)
-a : affichage de la syntaxe abstraite
-t : affichage de la table des symboles

Y est également un paramètre, correspondant au fichier
ou au chemin du fichier sur lequel vous voulez lancer le
compilateur.
